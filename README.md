# 妖刀记

**默默猴：**

    每個在華人世界長大的孩子，心中都有武俠夢。在那裡，籍籍無名的少年仗劍馳馬，自波瀾壯闊的冒險中成長茁壯，得到一些、也失去一些，最後立下不世功勳，成為英雄；其間，有慧美多情的紅顏知己、有義氣相投的生死兄弟、有城府深沈的反派梟雄，帝王將相，市井游俠，在故事裡起起落落……

    作夢的孩子終將長大，夢卻不曾消失。於是現在，我寫下了《妖刀記》。
